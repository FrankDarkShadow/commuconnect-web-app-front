Instrucciones

Para cargar sesiones presenciales masivamente, siga estos pasos:

  1. Complete la información de las sesiones respetando la estructura indicada
  2. Asegúrese de que los IDs de locales existan en el sistema
  3. Use el formato de fecha exacto: DD/MM/YYYY HH:MM
  4. Guarde el archivo y selecciónelo usando el botón "Seleccionar Archivo"
  5. Haga clic en "Cargar Sesiones" para procesar el archivo

Estructura del archivo Excel:

Campo Descripción Obligatorio
{{ campo.campo }} {{ campo.descripcion }} {{ campo.obligatorio ? 'SÍ' : 'NO' }}

Notas Importantes:

  • El archivo debe ser formato Excel (.xlsx o .xls)
  • La primera fila debe contener los nombres de las columnas exactamente como se especifica
  • El tipo de sesión se asigna automáticamente como "Presencial"
  • La fecha de fin debe ser posterior a la fecha de inicio
  • El local debe existir en la base de datos
  • Si no se proporciona descripción, se asigna "Sesión presencial" por defecto
  • Formato de fecha obligatorio: DD/MM/YYYY HH:MM (ejemplo: 15/03/2025 09:00)

Ejemplo de datos válidos:

id_local fecha_inicio fecha_fin capacidad descripcion
1 15/03/2025 09:00 15/03/2025 10:00 20 Yoga Básico
2 16/03/2025 14:00 16/03/2025 15:30
